# Icon sprite

All icons inherit `currentColor`, so their color can be changed in CSS.

```html
<svg width="24" height="24" aria-hidden="true">
  <use href="./icons/whatsvp-icons.svg#map"></use>
</svg>
```

Available IDs: `live-local`, `people`, `shield`, `map`, `nearby`, `events`, `updates`, `guilds`, `channels`, `meetup`, `polls`, `fees`, `fund`, `stories`, `live-rooms`, `host-tools`, `go-live`, `invite`, `screen-share`, `record`, `send`, `wallet`, `search`, `pin`, `message`, `plus`, `arrow`, `check`.
